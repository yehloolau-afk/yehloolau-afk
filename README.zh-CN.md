产品设计师，业余时间做 AI 工具。

我没有传统的工程背景——我用 Claude Code 和 coding agent 把设计想法变成真实可用的产品。对我来说，代码只是另一种设计材料。要把东西真正做出来上线的约束，比任何原型图都更能逼出好的思考。

我现在专注于让设计师变得更有能力的工具：降低从有想法到让用户用上之间的摩擦。

## 我怎么做东西

几乎所有东西都是单个 HTML 文件，部署在 GitHub Pages 上。

`Claude Code` · `原生 HTML / CSS / JS` · `GitHub Actions` · `Cloudflare Workers`

我首先是个设计师。「不用框架、不要服务器」的约束让东西保持简单和可交付。
